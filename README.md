# RestAPI - Node.js Backend 🚀

Uma interface robusta para manipulação e gestão de dados, construída com foco em performance, escalabilidade e documentação clara.

## 🛠️ Tecnologias Utilizadas
* **Runtime:** Node.js
* **Framework:** Express.js
* **Documentação:** Swagger / OpenAPI 3.0
* **Cloud/Deployment:** Azure Services
* **Validação:** Z-Schema / Validator.js

## 🌟 Funcionalidades
* **Arquitetura RESTful:** Implementação de rotas limpas e seguindo os métodos HTTP standards.
* **Documentação Interativa:** Interface Swagger integrada para teste de endpoints em tempo real.
* **CI/CD:** Workflows de GitHub Actions configurados para deployment automático em Azure.
* **Segurança:** Validação rigorosa de inputs e tratamento centralizado de erros.

## 🚀 Como Executar
1. Clone o repositório: `git clone https://github.com/JChorao/RestAPI.git`
2. Instale as dependências: `npm install`
3. Inicie o servidor: `npm start`
4. Aceda a `http://localhost:3000/api-docs` para ver o Swagger.
